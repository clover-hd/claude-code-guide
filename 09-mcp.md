## MCPとは

**MCP（Model Context Protocol）** は、Claude Codeに外部ツールやサービスを接続する仕組みです。「Claude CodeのUSB-Cポート」のようなもので、様々なツールをプラグインできます。

## 設定方法

プロジェクトルートの `.mcp.json` またはClaude Codeの設定ファイルで定義します。

## 実例：MCPサーバー構成

実際のプロジェクトで使用している**3つのMCPサーバー**の構成例です。

### 1. Sequential Thinking（構造的思考）

```
用途: 複雑な問題を段階的に分析・推論する
場面: アーキテクチャ設計、影響範囲の分析、意思決定
```

複雑な設計判断が必要なとき、Claudeが「まず整理してから考える」ための思考ツールです。

### 2. Serena（コードインテリジェンス）

```
用途: シンボルベースのコード解析・ナビゲーション
機能: シンボル検索、パターン検索、プロジェクトメモリ
場面: 大規模コードベースの構造理解、関数定義の追跡
```

TypeScriptのLSP（Language Server Protocol）と連携し、IDEのような精密なコード解析をClaude Codeに提供します。

### 3. Context7（ライブラリドキュメント）

```
用途: npmライブラリの最新ドキュメントを取得
機能: ライブラリID解決、ドキュメント取得
場面: 「Prismaの最新のマイグレーション方法は？」等の質問
```

Claudeの学習データにない最新のライブラリ情報を補完します。

## MCPの活用例

```
あなた: 「Stripeの決済フローを実装したい」

Claude Code の動き:
  1. Context7 → Stripe SDKの最新ドキュメントを取得
  2. Serena → 既存のコードベースで関連ファイルを特定
  3. Sequential Thinking → 実装方針を段階的に整理
  4. 実装開始
```

## よく使われるMCPサーバー

| MCPサーバー | 用途 |
|------------|------|
| **sequential-thinking** | 複雑な問題の構造化思考 |
| **serena** | コードベースの深い解析 |
| **context7** | ライブラリドキュメント取得 |
| **filesystem** | ファイルシステム操作 |
| **github** | GitHub APIとの連携 |
| **postgres / mysql** | データベース直接操作 |
