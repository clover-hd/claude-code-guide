# 最初の30分チュートリアル

[< ガイド目次に戻る](README.md)

---

インストールが終わったら、まずはこのチュートリアルで「AIと一緒にコードを書く体験」をしてみましょう。**読むより先に手を動かす**のが一番です。

## 準備

```bash
# 1. 作業用ディレクトリを作成
mkdir my-first-claude-project && cd my-first-claude-project

# 2. gitリポジトリを初期化（Claude Codeはgitがあると賢く動く）
git init

# 3. Claude Codeを起動
claude
```

起動すると、以下のようなプロンプトが表示されます：

```
╭──────────────────────────────────────╮
│ ✻ Welcome to Claude Code!           │
│                                      │
│ /help for available commands         │
╰──────────────────────────────────────╯

>
```

これで準備完了です。

---

## Step 1：まずは会話してみる（1分）

何でもいいので話しかけてみましょう。

```
> 自己紹介して
```

Claudeが返答します。これだけで「ターミナルでAIと対話できる」ことがわかります。

---

## Step 2：ファイルを作らせてみる（3分）

次に、Claudeにコードを書かせてみます。

```
> シンプルなHTMLファイルを作って。
  ボタンを押すとカウントが増えるカウンターアプリ。
  デザインはモダンな感じで。
```

Claudeが `index.html` を作成しようとします。**承認プロンプトが表示されます**：

```
Claude wants to create index.html
Allow? [y/n/a]
```

`y` を押して許可しましょう。ファイルが作成されます。

```bash
# ブラウザで開いて確認（Claude Codeを終了せずに別ターミナルで）
open index.html        # macOS
xdg-open index.html    # Linux
explorer.exe index.html # WSL
```

**ブラウザにカウンターアプリが表示されたら成功です。**
これが「自然言語で指示するだけでコードが生まれる」体験です。

---

## Step 3：修正を指示してみる（3分）

次に、作ったものを修正させてみましょう。

```
> カウンターに以下の機能を追加して：
  - リセットボタン
  - カウントが10以上になったら数字を赤くする
  - ダークモード対応
```

Claudeがファイルを編集します。再度ブラウザで確認してみてください。「追加の指示だけで機能が増える」感覚を体験できます。

---

## Step 4：質問してみる（2分）

Claudeはコードの説明もできます。

```
> このindex.htmlのコードを初心者にもわかるように解説して
```

自分が書いていないコードでも、Claudeに聞けば理解できます。これは既存プロジェクトに参加するときにも使えるテクニックです。

---

## Step 5：もう少し複雑なものに挑戦（10分）

ここからは自由です。以下のどれかを試してみてください。

### 案A：ToDoアプリ

```
> ToDoアプリを作って。HTML + CSS + JavaScriptで。
  機能：
  - タスクの追加・削除・完了チェック
  - LocalStorageで永続化（リロードしても消えない）
  - シンプルでおしゃれなUI
```

### 案B：じゃんけんゲーム

```
> ブラウザで遊べるじゃんけんゲームを作って。
  - グー・チョキ・パーのボタンで選択
  - コンピュータの手はランダム
  - 勝敗の記録を表示
  - アニメーション付きで楽しい感じに
```

### 案C：自分のプロフィールページ

```
> エンジニアのポートフォリオページを作って。
  名前: （あなたの名前）
  スキル: Java, Python, etc（あなたのスキル）
  レスポンシブ対応で、モダンなデザインにして。
```

---

## Step 6：Plan Modeを試す（5分）

作ったアプリに機能を追加する前に、Plan Modeで計画を立ててみましょう。

```
# Shift+Tab を押す（画面下部に「⏸ plan mode on」と表示）

> このToDoアプリにカテゴリ機能を追加したい。
  「仕事」「プライベート」「買い物」のカテゴリで分類できるようにしたい。
  実装計画を立てて。
```

Claudeがコードを読んで、**何をどう変更するか**の計画を提示してくれます。
計画に納得したら `Shift+Tab` でPlan Modeを解除し、「この計画で実装して」と指示します。

---

## Step 7：コミットしてみる（2分）

最後にgitコミットをClaude Codeにやらせてみましょう。

```
> 今の変更をgitコミットして。
  コミットメッセージは日本語で、何を作ったかわかるようにして。
```

Claudeがステージング → コミットメッセージ作成 → コミットまで実行します。

| 体験 | 学んだこと |
|------|-----------|
| **会話** | Claude Codeはターミナルで動くAIパートナー |

| **修正指示** | 追加の要望を伝えるだけで機能が増える |
| **コード解説** | 他人のコード（AIのコード）も聞けば理解できる |
| **Plan Mode** | 作る前に計画を立てると品質が上がる |
| **コミット** | gitの操作もClaude Codeがやってくれる |
